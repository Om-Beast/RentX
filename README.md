# RentX 🚗

RentX is a full-stack vehicle rental platform built using the MERN stack.

## Features

- Browse available vehicles
- Vehicle details page
- Rental booking flow
- Checkout page
- Booking confirmation page
- MongoDB integration
- Responsive UI

## Tech Stack

### Frontend
- React.js
- React Router
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB
